# FlutterDemo

Flutter 各种Demo实现效果：

1：2种 仿朋友圈 效果,顶部拉伸 和 不拉伸

2：仿抖音上下滑动视频播放

3：视频直播（使用的电视台的m3u8格式 直播）

4：拍照和录像功能

5：拟物化UI

6：俄罗斯方块

7：Google chrome 断网 小恐龙彩蛋游戏

8：3D 效果实现

9: App内全局 悬浮窗视频播放

10: 各种炫酷背景效果
